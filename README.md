본 저장소는 「청년층 자동차 소유 감소와 이동 소비 패턴 변화 분석」을 위한 프로젝트 코드, 데이터, 분석 결과를 포함하고 있습니다. 최근 10여 년간 20·30대의 자동차 소유율이 지속적으로 감소하고 있다는 사회적 관찰을 데이터 기반으로 검증하고, 그 배경을 실질임금 변화, 공유차량 확산, 소비자물가지수 등의 지표를 통해 구조적으로 설명하는 것을 목표로 합니다.
본 분석은 2012년부터 2025년까지의 시계열 데이터를 통합하여, 청년층의 이동 방식이 “자가용 중심”에서 “공유·대체 이동수단 중심”으로 전환되는 흐름을 정량적으로 평가합니다.

분석에는 네 가지 주요 데이터가 사용되었습니다: 자동차등록현황(성별·연령별), 소비자물가지수(CPI), 카셰어링 등록 차량 수, 연령별 임금 데이터입니다. 각 데이터는 관측 연도와 제공 방식이 서로 달랐기 때문에 직접적으로 비교가 불가능했습니다. 이를 해결하기 위해 모든 데이터를 2012~2025년 기준의 단일 시계열로 통일하였고, 결측값에 대해서는 선형 보간과 회귀 기반 외삽을 적용하여 분석에 필요한 완전한 데이터셋을 생성하였습니다.

프로젝트의 핵심 분석은 Jupyter Notebook(notebook.ipynb)에서 수행되었습니다. 여기에는 데이터 전처리, 변수 생성, 예측 모델, 시각화 코드가 블록 단위로 정리되어 있어, 전체 분석 흐름을 단계별로 확인할 수 있습니다. 프로젝트에 포함된 data/ 폴더에는 분석에 사용된 CSV 자료가 저장되어 있으며, 재현성을 위해 코드에서 절대경로가 아닌 상대경로를 사용하였습니다.

실행 환경은 Python 3.10 이상이며, pandas, numpy, matplotlib, scikit-learn을 포함한 필수 패키지는 requirements.txt를 통해 자동 설치할 수 있습니다. 분석 재현을 원할 경우 저장소를 클론한 뒤, Jupyter Notebook 환경에서 final.ipynb를 실행하면 됩니다.

본 프로젝트는 단순한 수치 비교를 넘어, 청년층 자동차 구매 감소가 구조적인 경제 요인(실질임금 정체), 도시 모빌리티 대안의 확산(카셰어링 증가), 자동차 유지비 관련 물가 상승 등 여러 요인과 함께 나타나는 복합적인 현상임을 데이터 기반으로 보여줍니다. 또한, 코드 및 데이터가 모두 공개되어 있어 누구나 동일한 분석을 재현하거나 확장할 수 있도록 구성되어 있습니다.
